    normalize_path(path) click to toggle source

    Normalizes URI path.

    Strips off trailing slash and ensures there is a leading slash.

    normalize_path("/foo")  # => "/foo"
normalize_path("/foo/") # => "/foo"
normalize_path("foo")   # => "/foo"
normalize_path("")      # => "/"

    # File lib/journey/router/utils.rb, line 14
def self.normalize_path(path)
  path = "/#{path}"
  path.squeeze!('/')
  path.sub!(%r{/+\Z}, '')
  path = '/' if path == ''
  path
end
